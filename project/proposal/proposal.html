Project Proposal

Zachary Moring

Using PHP and Laravel, I will construct an application for my nonprofit's accounting. In particular, it will allow for the following:

  1. Allow the user to define (and persist with a database) account information for double-entry ledger accounting: Name, account number, etc.
  2. Enter individual expenses against an account and counter-account;
  3. For detail view for an account, produce a list of all transactions for that account, including the running sum, etc.
